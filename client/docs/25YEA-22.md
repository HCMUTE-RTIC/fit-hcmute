# Task 25YEA-22: [FE_Admin] CMS Article Module

## 1. Overview

This task involves building the CMS Article module for the 25th Anniversary Website of FIT-HCMUTE. It includes the following features:

- Article List Page.
- Article Editor with CKEditor integration.
- Meta SEO Configuration Form.

## 2. Technical Requirements

- **Framework**: Next.js (App Router).
- **Styling**: Tailwind CSS, MagicUI.
- **Rich Text Editor**: CKEditor 5.
- **Design Inspiration**: TailAdmin, Figma.
- **SEO Rules**: Automated slug generation, Meta Title/Description/Keywords validation.

## 3. Implementation Plan (Step-by-Step)

1. **Admin Layout Setup**: Side-bar and Header implementation.
2. **Article List Page**: Data table with filtering and pagination.
3. **Article Editor**:
   - Integration of CKEditor.
   - Article form with Title, Content, Status, Thumbnail.
   - SEO Meta Configuration section.
4. **Verification & Polish**: Visual audit and flow testing.

### [2026-02-27] Bước 2.1: Sửa lỗi khởi tạo CDN EditorJS

- **Vấn đề phát sinh**: Next.js báo lỗi `Cannot read properties of undefined` khi init Plugins và EditorJS vì các link CDN `@latest` trỏ về file chưa được đóng gói chuẩn (bundle) hoặc do script trên Window chưa load kịp.
- **Giải pháp áp dụng**:
  1. Trỏ chính xác đến các file `umd` và `bundle.js` (`editorjs.umd.min.js`, `dist/bundle.js`) thay vì gọi root package để trình duyệt có thể nhúng thư viện chuẩn.
  2. Thêm hàm `script.onerror` trong luồng nhúng Script. Nếu tải Script CDN thất bại (rớt mạng gốc), tiến trình Promise `checkReady` vẫn được đi tiếp mà không làm treo React App.
  3. Sửa cấu trúc Injection Tool của EditorJS. Sử dụng Rest Operator và Conditional Loading: `...(window.Header ? { header: { class: window.Header } } : {})` để đảm bảo việc đọc class Object từ thuộc tính Window là an toàn tuyệt đối và bỏ qua rủi ro khi một Plugin CDN bảo trì/bị lỗi load.
